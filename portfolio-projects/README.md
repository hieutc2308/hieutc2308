# Portfolio Projects

Working area for portfolio case-study source material.

Each project folder mirrors a project slug used by the website route:

- `healthtech-clinic-analytics`
- `powerbi-admin-monitoring`
- `fashion-retail-analytics`
- `bpo-workforce-analytics`
- `employee-turnover-analysis`

Suggested structure inside each project:

- `assets/` - screenshots, exported visuals, diagrams, and images
- `documents/` - briefs, case-study drafts, PDF exports, and reference docs
- `notes/` - implementation notes, metric definitions, and decisions
- `repo/` - repo links, code pointers, sanitized snippets, or setup notes
